# 介绍

* 此私有协议专用于基于 TCP 协议开发应用程序的开发人员。有关 UFACTORY Studio 应用程序开发，请参阅相应的产品用户手册。
* 考虑到使用协议进行应用开发的潜在风险，工作人员需要阅读并理解相应产品用户手册的所有内容，熟悉风险评估和机器人运动规划，并在开发前熟练掌握“UFACTORY Studio”中的机器人参数设置和程序创建。
