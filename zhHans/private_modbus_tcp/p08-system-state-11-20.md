# 系统状态(11-20)

## 使能/关闭伺服(系统重置)

**寄存器:11(0x0B)**

**警告**:</br>

以上操作会使正在进行的运动终止并清空指令缓存，表现与 STOP 状态相同。

```
// 请求：
00 01 00 02 00 03 0B 08 01
```



> 请求指令说明\
> //00 01    U16, 事务标识\
> //00 02    U16, 协议\
> //00 03    U16, 长度\ 
> //0B       U8, 寄存器\
> //08       U8</br>\ 
>参数 1 关节号（选择所有关节）</br>\
1-7：选择某一个关节</br>
8：选择所有关节</br>
> //01       U8,</br>\
>参数 2 （使能机械臂）是否使能</br>\
1：为使能（0x01）</br>
0：为关闭电机（0x00）





```
// 响应:
00 01 00 02 00 02 0B 10
```



> 响应指令说明\
> //00 01    U16, 事务标识\
> //00 02    U16, 协议\
> //00 02    U16, 长度\ 
> //0B      U8, 寄存器\
> //10      U8, 状态

## 设置系统运动模式(系统重置)

**寄存器:19 (0x13)**

**警告**:</br>
以上操作会使正在进行的运动终止并清空指令缓存，表现与 STOP 状态相同。

```
// 请求：
00 01 00 02 00 03 13 00 00
```



> 请求指令说明\
> //00 01    U16, 事务标识\
> //00 02    U16, 协议\
> //00 03    U16, 长度\
> //13       U8, 寄存器\
> //00       U8, 运动模式</br>
0 ：位置控制模式</br>
1 ：servo 运动模式</br>
2 ：关节示教模式</br>
3 ：笛卡尔示教模式（暂无）</br>
4：关节速度模式</br>
5：笛卡尔速度模式</br>
6：关节在线规划模式</br>
7：笛卡尔在线规划模式</br>
> //00       U8, （示教模式负载检测）
0：开启
1：关闭



```
// 响应:
00 01 00 02 00 02 13 00
```



> 响应指令说明\
> //00 01    U16, 事务标识\
> //00 02    U16, 协议\
> //00 02    U16, 长度\
> //13       U8, 寄存器\
> //00       U8, 状态
