# 485外设透传(240-241)

## 外设 485 透传超时

**寄存器:240(0xF0)**

```
// 请求:
00 01 00 02 00 02 F0 01 
```



> **请求指令说明**\
> //00 01    U16, 事务标识\
> //00 02    U16, 协议\
> //00 02    U16, 长度\
> //F0       U8, 寄存器\
> //01       U8, 超时(s)



```
// 响应:
00 01 00 02 00 02 F0 00
```



> **响应指令说明**\
> //00 01    U16, 事务标识\
> //00 02    U16, 协议\
> //00 02    U16, 长度\
> //F0       U8, 寄存器\
> //00       U8, 状态



## 外设 485 透传通信

**寄存器:241(0xF1)**

```
// 请求:
00 01 00 02 00 0B F1 09 00 
```



> **请求指令说明**\
> //00 01    U16, 事务标识\
> //00 02    U16, 协议\
> //00 0B    U16, 长度\
> //F1       U8, 寄存器\
> //09       U8, 主机ID, 末端485： 0x09, 控制器485：0x0A.\
> //00       用户数据


```
// 响应:
00 01 00 02 00 08 F1 00 09 00
```



> **响应指令说明**\
> //00 01    U16, 事务标识\
> //00 02    U16, 协议\
> //00 08    U16, 长度\
> //F1       U8, 寄存器\
> //00       U8, 状态\
> //09       U8, 主机ID, 末端485： 0x09, 控制器485：0x0A\
> //00       用户数据
