# v2.6.0
发布时间：2025.04.10  
如果设备信息页面的机械臂SN一栏为空，请不要升级并联系技术支持。

## 固件 V2.6.0
1. 自定义自碰撞几何模型支持X/Y/Z 方向的偏移。
2. 新增六维力矩传感器拖动示教的启动力与启动力矩的读写接口。
3. 修复笛卡尔运动参数缺省导致的规划轨迹异常问题。
4. 修复交融半径小于1mm 或者超过两点线段长度导致规划轨迹异常的问题。
5. 修复带圆弧的直线运动圆弧转直线处速度不连续的问题。
6. 修复若干问题。

## Studio V2.6.0
1. 实时控制-兼新的末端执行器"BIO 机械爪G2"。
2. 实时控制-自定义末端执行器几何模型支持长方体/圆柱模型的X/Y方向偏移显示。
3. Blockly-编辑点位页面增加移动按钮，长按移动到当前点位。
4. Blockly-增加获取单个关节角度和TCP位置的代码块。
5. Blockly-位置触发IO的位置和IO编号和值支持传入变量。
6. Blockly-数字输出增加队列执行（Sync）参数，默认值为True
7. Blockly-新增"BIO机械爪 G2"控制代码块，允许设置位置/速度/力。
8. Python IDE-首页增加Python IDE。
9. Python IDE-允许删除BlocklytoPython文件夹内的文件。
10. Python IDE-新增独立的Python IDE页面，浏览器输入"控制盒IP:18333/ide" 访问。
11. 设置-调参工具，限制CSV 最大记录行数为1000,000 行，限制保存文件数最多为5个。
12. 设置-外接设备，Modbus TCP 支持UFACTORY 私有协议.
13. 设置-辅助功能，增加快捷输入开关，开启后可在实时控制页面输入关节角和位置。
14. 设置-辅助功能，增加网络检测开关。
15. 设置-通用设置-高级设置，增加只读模式开关，开启后不允许编辑Blockly/Gcode/Python IDE 的工程文件。
16. 报错处理-优化S17的错误处理,增加S54,S55的错误处理。
17. 优化在旧版本iPad操作系统上的显示。
18. 修复若干问题。
