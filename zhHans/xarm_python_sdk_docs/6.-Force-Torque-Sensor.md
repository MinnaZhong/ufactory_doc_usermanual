# 6. 力矩传感器 (Force Torque Sensor)

对于官方的六维力矩传感器，SDK提供了一系列接口进行控制和数据读取。

| API函数 | 描述 |
| :--- | :--- |
| `set_ft_sensor_enable()` | 使能或断使能力矩传感器 |
| `get_ft_sensor_data()` | 获取力矩和力的数据 |
| `ft_sensor_set_zero()` | 将当前状态设为零点 |
| `ft_sensor_app_set()` | 设置力控应用类型 (如力控或混合力位控) |
| `ft_sensor_app_get()` | 获取当前力控应用类型 |
| `set_ft_collision_detection()` | 设置基于力矩传感器的碰撞检测 |
| `set_ft_collision_threshold()` | 设置力矩传感器碰撞检测的阈值 |
| `set_ft_collision_rebound()` | 设置碰撞后是否回弹 |
| `set_ft_collision_reb_distance()` | 设置碰撞后回弹的距离 |

## 获取数据

使用 `get_ft_sensor_data(is_raw=False)` 方法可以获取传感器数据。
- `is_raw`: 布尔值。默认为 `False`，返回经过重力补偿和滤波后的数据；如果为 `True`，则返回原始数据。
- 返回值: 一个包含错误码和数据数组的元组。数据数组中包含了Fx, Fy, Fz, Tx, Ty, Tz的值。

**示例：**

```python
# 1. 使能力矩传感器
arm.set_ft_sensor_enable(1)
time.sleep(0.5)

# 2. 将当前状态设为零点
arm.ft_sensor_set_zero()
time.sleep(0.5)

# 3. 获取数据
code, data = arm.get_ft_sensor_data()
if code == 0:
    print("力矩传感器数据:", data)

# 4. 断使能力矩传感器
arm.set_ft_sensor_enable(0)
```

## 碰撞检测示例

下面是一个使用力矩传感器进行碰撞检测的简单示例。机械臂会向下运动，直到接触桌面并触发碰撞检测，然后回弹。

```python
# 1. 初始化并运动到一个安全位置
arm.motion_enable(enable=True)
arm.set_mode(0)
arm.set_state(0)
arm.set_position(x=250, y=0, z=300, roll=-180, pitch=0, yaw=0, speed=50, wait=True)

# 2. 使能力矩传感器并清零
arm.set_ft_sensor_enable(1)
time.sleep(0.5)
arm.ft_sensor_set_zero()
time.sleep(0.5)

# 3. 设置碰撞检测参数
# 设置Z轴方向的力阈值为10N，其他方向阈值可以设得很大以忽略
arm.set_ft_collision_threshold(tool_axis=[30, 30, 10, 30, 30, 30]) 
# 设置碰撞后回弹
arm.set_ft_collision_rebound(on=True)
# 开启碰撞检测
arm.set_ft_collision_detection(on=True)
time.sleep(0.5)

# 4. 执行可能触发碰撞的运动
print("开始向下运动，直到检测到碰撞...")
# 让机械臂在Z轴方向向下运动，直到触发碰撞
code = arm.set_position(x=250, y=0, z=100, roll=-180, pitch=0, yaw=0, speed=20, wait=True)
print(f"运动完成，返回码: {code}")
if arm.has_err_warn:
    print("检测到错误或警告，可能已触发碰撞。")
    arm.clean_error()

# 5. 关闭碰撞检测并断使能力矩传感器
arm.set_ft_collision_detection(on=False)
arm.set_ft_sensor_enable(0)

print("碰撞检测示例完成。")

```
**注意：** 在使用前，请确保力矩传感器已经正确安装并已在UFACTORY Studio中启用。
