# 12.set_multiple_holding_register
设置多个保持寄存器地址的值。

### Message:  
```json
# 设置多个保持寄存器地址的值
{
    "cmd": "set_multiple_holding_register",
    "data": {
        "addr": "0x0003",
        "value_li": ["0x04D2", "0x0D80"]
    },
    "id": "1"
}
```
### Request：  
* cmd
  * string，必填，请求接口
* date
  * json，必填，请求数据
* id
  * string，必填，请求序列号
 
    
| **data** | **数据类型** | **必须** | **描述**     | **示例值** |
| -------- | ------------ | -------- | ------------ | ---------- |
| addr     | String       | 是       | 设置的地址值 | 0x0100     |
| value_li    | Array       | 是       | 需要设置多个值的数组 | ["0x04D2", "0x0D80"]        |

### Response:  
```
code=0->成功；
code!=0->失败，详情参考xArm_Python_SDK/xarm_api_code.md文档。
```

#### 请求参数
无
