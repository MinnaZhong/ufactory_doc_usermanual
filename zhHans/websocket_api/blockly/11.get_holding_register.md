# 11.get_holding_register
获取保持寄存器地址的值。

### Message:  
```json
# 获取保持寄存器地址的值
{
    "cmd": "get_holding_register",
    "data": {
        "addr": "0x0020",
        "quantity": "2"
    },
    "id": "1"
}
```
### Request：  
* cmd
  * string，必填，请求接口
* date
  * json，必填，请求数据
* id
  * string，必填，请求序列号
 
    
| **data** | **数据类型** | **必须** | **描述**     | **示例值** |
| -------- | ------------ | -------- | ------------ | ---------- |
| addr     | String       | 是       | 获取的地址值 | 0x0020     |
| quantity    | String       | 是       | 需要获取多少个寄存器的值 | 2      |

### Response:  
```
code=0->成功；
code!=0->失败，详情参考xArm_Python_SDK/xarm_api_code.md文档。
```

#### 请求参数
无
